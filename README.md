# tg-bot-factory

Telegram-бот-конструктор, который помогает проектировать и генерировать других Telegram-ботов.

## Что это

Этот проект задуман как Telegram-интерфейс + генератор спецификаций + шаблоны проектов + GitHub-синхронизация.

Пользователь описывает задачу:
- FAQ-бот
- бот-заявки
- бот-запись
- AI-бот
- контент-бот
- бот с админкой

Система:
1. определяет тип бота
2. собирает спецификацию
3. предлагает архитектуру
4. генерирует структуру проекта
5. подготавливает код
6. сохраняет изменения без полной перегенерации проекта

## Главный принцип

Если пользователь просит изменить только одну часть проекта, система меняет только нужные файлы и не пересобирает всё целиком.

## План MVP

### Этап 1
- базовый Telegram-бот на `aiogram`
- главное меню
- режимы: Create / Edit / Review / Template
- сбор спецификации

### Этап 2
- шаблоны ботов
- генерация структуры проекта
- генерация файлов

### Этап 3
- GitHub sync
- точечное редактирование файлов
- audit режима
