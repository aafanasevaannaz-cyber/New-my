# Instagram-бот: план реализации

## Что делает бот

1. Ты присылаешь сценарий рилса или поста.
2. Бот ищет в тексте trigger-слово.
3. Обычно это слово стоит после формулировок:
   - "пиши ..."
   - "напиши ..."
   - "в комментариях пиши ..."
   - "оставь слово ..."
4. Бот сохраняет правило для конкретного контента.
5. Потом бот отслеживает комментарии.
6. Если человек пишет нужное слово, бот отправляет сообщение в личку.

## Что бот должен найти в тексте

- trigger-слово
- что обещано пользователю
- короткий ответ в комментарии
- текст личного сообщения

## Как реализуем

### Шаг 1. Анализ текста
Бот получает текст и вытаскивает:
- слово после "пиши" или похожей формулировки
- обещанный материал
- шаблон ответа

### Шаг 2. Сохранение правила
Правило хранится так:
- какой рилс / пост
- какое слово
- что отправлять
- какой текст в личку

### Шаг 3. Отслеживание комментариев
Когда приходит комментарий, бот:
- сравнивает текст комментария со словом
- если совпало, запускает отправку

### Шаг 4. Отправка в личку
Бот отправляет заранее подготовленный текст, ссылку или другой допустимый формат.

## Что нужно кроме GitHub

GitHub сам по себе не запускает такого бота.
Нужны ещё:
- backend, где будет жить логика бота
- подключение к Instagram / Meta
- место, где бот будет постоянно работать
- база для хранения правил и логов

## Удобный минимум

- GitHub — хранить проект
- Python backend — логика
- сервер / хостинг — чтобы бот работал постоянно
- Meta / Instagram integration — чтобы получать комментарии и отправлять ответы
- база данных — чтобы помнить правила и не слать по кругу

## Важно

Бот не придумывает слово случайно.
Он ищет его прямо в твоём сценарии.
